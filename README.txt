# Mobile Legends Account Checker - Installation Guide

## Installation Instructions (Termux)

1. **Update & Upgrade Termux**  
   Run the following command to ensure Termux is up-to-date:  
   ```sh
   pkg update && pkg upgrade
   ```

2. **Install Node.js**  
   Install Node.js to run the script:  
   ```sh
   pkg install nodejs
   ```

3. **Clone the Repository**  
   Download the script by cloning the repository:  
   ```sh
   git clone https://github.com/yourusername/mlbb-valid-checker.git
   cd mlbb-valid-checker
   ```

4. **Install Dependencies**  
   Install necessary dependencies:  
   ```sh
   npm install axios fs readline
   ```

## Notes:
- Ensure you have a **stable internet connection**.
- If `node` is not recognized, restart Termux and try again.
